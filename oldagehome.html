Old Age Home Management System

Developed an Old Age Home Management System as my UG final year project using HTML, CSS, PHP, and SQL. The project was designed to manage and organize old age home activities digitally through a simple and user-friendly web application.

The frontend of the project was developed using HTML and CSS to create a clean, responsive, and easy-to-use interface. PHP was used for backend development to handle application logic, form processing, data management, and user interactions efficiently.

SQL database was used to store and manage resident details, staff information, medical records, room allocations, visitor details, and other important data securely. The system allows administrators to easily add, update, delete, and manage records through the web interface.

The project mainly focused on improving data management, reducing manual paperwork, maintaining resident information properly, and making the management process more efficient and organized.

Through this project, I improved my knowledge in frontend development, backend integration, database connectivity, CRUD operations, responsive web design, and practical implementation of web application development using HTML, CSS, PHP, and SQL.

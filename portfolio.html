Personal Portfolio Website

Developed a Personal Portfolio Website using HTML and CSS as part of my internship learning activities. The project was created to showcase my personal profile, technical skills, projects, education details, and contact information through a professional web interface.

The portfolio website was designed with a clean, responsive, and modern user interface using only HTML and CSS. Different sections such as Home, About Me, Skills, Projects, Education, Resume, Social Links, and Contact were structured properly to create a complete personal portfolio experience.

The project mainly focused on improving frontend development skills, responsive web designing techniques, layout structuring, animations, card designs, navigation systems, and UI alignment concepts.

Interactive elements such as hover effects, smooth scrolling navigation, animated sections, responsive cards, and professional color combinations were implemented to make the website visually attractive and user-friendly.

Through this project, I improved my practical knowledge in HTML and CSS, website structuring, responsive design implementation, and modern frontend UI development concepts while building my own professional portfolio website.

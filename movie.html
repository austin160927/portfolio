Movie Website

Developed a responsive Movie Website using HTML and CSS during my UG internship to provide a simple and interactive movie browsing experience for users. The website was designed to organize and display movies year-wise, making it easy for users to explore movies based on their release year. Each movie section contains movie posters, titles, and interactive buttons that allow users to click and watch the selected movie or related video content.

The project mainly focused on creating a clean user interface, smooth navigation, responsive layouts, and visually attractive movie cards using only HTML and CSS. Different sections were designed for various movie categories and years to improve user experience and accessibility. Hover effects, buttons, layouts, and responsive designs were implemented to make the website more modern and user-friendly.

Through this project, I improved my frontend development skills, responsive web design techniques, UI structuring knowledge, and practical implementation of HTML and CSS concepts. This project also helped me understand website layouts, alignment techniques, styling methods, and how to create visually appealing interfaces without using advanced frameworks or JavaScript.
